# cx-studio-tk

## 介绍

此工具包是一套用于优化影视后期工作的工具集。

此工具包包含三个互相依赖的部分：

1. cx-studio
   是一个提供相关基础设施的小型类库。你可以单独使用它开发你自己的工具。

2. cx-wealth
   是一个包含多种命令行控件的库，是 rich 的扩展。

3. cxalio-studio-tools
   是一套可以直接使用的命令行工具。其中的工具也可以作为依赖库使用，单独调用其中的功能。
   这个工具包依赖前面的两个库。

## 安装方法

直接使用 pip 安装即可。

建议使用 pipx 等独立环境的工具安装 cxalio-studio-tools。
